# Pixel Text (原点短信)

一款纯粹的、符合原生 Android 审美的短信应用。在不牺牲隐私的前提下，通过本地智能解析技术，将枯燥的验证码、服务短信转化为精美的
Material You 卡片，致力于成为 Google Pixel 系列在国内的完美默认短信替代品。

## ✨ 核心特性 (Features)

- 🔒 **绝对隐私安全**: 应用**无任何网络权限**，所有的短信读取与智能解析均在本地设备上完成，确保您的数据零泄露。
- 🧩 **本地化智能解析**: 针对国内常用的 12306 购票短信、各类验证码进行专门的正则解析，提取核心信息并转化为直观的交互卡片（支持一键复制等操作）。
- 🎨 **Material You 审美**: 全面采用 Material 3 设计规范，支持 Android 动态取色（Dynamic
  Color），完美融入系统原生界面，并包含丝滑的原生过渡动效。
- 📱 **默认短信应用支持**: 完整实现了 Android 默认短信应用所需的核心组件，可以无缝接管系统短信收发。
- 📶 **双卡双待支持**: 智能识别短信接收卡槽，并在发送/回复时支持快速切换 SIM 卡。

## 🛠️ 技术栈 (Tech Stack)

- **语言**: Kotlin
- **UI**: Jetpack Compose (Material 3)
- **架构**: MVVM + Clean Architecture
- **异步处理**: Coroutines & Kotlin Flow

## 📋 模块划分与后续计划

当前项目处于 MVP（最小可行性产品）开发阶段，主要规划如下：

1. **基础短信服务**: 实现短信收发、监听新短信入库功能。
2. **消息列表流**: 通过 `ContentResolver` 实现响应式的短信会话展示。
3. **解析引擎接入**: 实现验证码、12306 行程信息的本地正则提取与卡片化 UI 渲染。
